# haarlem-bells

Evening ringing of the **Damiaatjes**, Grote Kerk (St. Bavo), Haarlem NL,
2026-07-17 — an ambisonic field recording and its full rhythmic/soundscape
analysis with [ambiscape](https://github.com/fourMs/ambiscape).

- **`damiaatjes-analysis.ipynb`** — the analysis as an executable notebook
- **`RHYTHM_REPORT.md`** — the full technical report (figures 1–10)
- **`blog-damiaatjes.md`** — accessible write-up with the history
- **`analysis/`** — ambiscape outputs (figures, rhythm.json, ISO indicators)
- **`annotations.json`** — Schaeffer/Schafer taxonomy annotations
- **`calibration.json`** — recorder clock correction (+665 s)
- **`deposit/`** — Freesound upload notes (audio not in repo)

The audio (`260717_001.WAV`, 847 MB, and the trimmed `deposit/*.wav`) is not
tracked here — the trimmed version is published on Freesound; place the
original WAV in the repo root to re-run the notebook end-to-end.
